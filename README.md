# CSU Bookstore Online Retail Toolkit

The CSU Bookstore Online Retail Toolkit is a collection of lightweight web-based tools designed to support eCommerce, fulfillment, customer communication, and operational workflows for the Colorado State University Bookstore Online Retail department.

The project is hosted through GitHub Pages and provides staff with quick access to commonly used tools, resources, and reference materials from a single dashboard.

## Purpose

This toolkit was developed to reduce repetitive manual work, standardize department processes, and improve consistency across online retail operations.

Many of the tools replace tasks that were previously completed using handwritten notes, Microsoft Word templates, spreadsheets, or manually formatted emails.

## Features

### Mini Apps

* **Product Listing Builder** – Generates product description HTML for the bookstore website.
* **Tracking CSV Builder** – Creates tracking upload files for shipments and orders.
* **Banner Ad Builder** – Creates structured homepage banner content.
* **Carousel Builder** – Builds responsive carousel content for website updates.
* **Communication Alert Builder** – Generates copy-and-paste HTML for CSU Bookstore communication alerts with preview, icons, scheduling, and display options.
* **Customer Email Builder** – Generates customer-ready email templates and matching Insite notes.
* **Email Builder Template Tool** – Creates reusable email templates used by Online Retail staff.
* **Phone Order Tax Allocator** – Splits phone-order sales tax across items proportionally while preserving exact rounding totals.
* **Order Range Sign Generator** – Produces professional shelf and rack signs for order retrieval areas.
* **Pickup Date Label Generator** – Creates reminder and restock labels for pickup orders.
* **Oversized / Secure Item Tag Generator** – Produces retrieval tags and documentation for oversized or secured merchandise.
* **Dot Date Label Generator** – Creates printable date labels for operational workflows.

### Artificial Intelligence

* **RAMBO (Beta)** – Resource for Assisting Merchandising and Backend Operations.
* **Online Retail Description Bot (Beta)** – Assists with product listing creation using CSU-approved AI tools.
* **CSU GPT** – Access to Colorado State University's approved AI platform.

## Security & Data Handling

This project is intentionally lightweight and does not connect directly to CSU, MBS, or other institutional systems.

The toolkit:

* Does not store customer information.
* Does not access student records.
* Does not connect directly to bookstore databases.
* Does not retrieve data from CSU or MBS systems.
* Operates entirely within the user's browser unless otherwise noted.

All generated content is created locally on the user's device.
